# E‑Commerce Data Analysis Project

## 📌 Overview
This project focuses on cleaning and analyzing an e‑commerce dataset using **Python** and popular data analysis libraries such as **Pandas, NumPy, Matplotlib, and Seaborn**.  
The goal is to demonstrate reproducible workflows for data cleaning, exploratory data analysis (EDA), and visualization.

## 📂 Dataset
The dataset contains transactional records with the following columns:

- **ID**  
- **Customer_Name**  
- **Order_ID**  
- **Order_Date**  
- **Product**  
- **Category**  
- **Quantity**  
- **Price**  
- **Payment_Method**  
- **Status**  
- **Revenue**

## 🛠️ Tools & Libraries
- **Python 3.x**
- **Pandas** – data cleaning & wrangling  
- **NumPy** – numerical operations  
- **Matplotlib & Seaborn** – data visualization  

## 🔧 Data Cleaning Steps
- Handling missing values  
- Removing duplicates  
- Converting data types (e.g., dates, numeric fields)  
- Standardizing categorical values (e.g., payment methods, status)  
- Creating new features (e.g., calculating revenue per order)

## 📊 Exploratory Data Analysis (EDA) 
- Category‑wise sales distribution  
- Revenue trends over time  
- Payment method preferences  
- Order status breakdown  

## 📈 Visualizations
- Bar charts for product categories  
- Line plots for revenue trends  
- Box plot for finding out outliers
- Pie charts for visualizing category and Revenue distribution
